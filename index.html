About The Project

Physical violence in both the public and private sectors has a significant impact on the quality and safety of people's lives. Owl Security System proposes an automatic physical violence detection system that could recognize violent behaviors. The system will send automated security alerts to security guards, and system configuration will be handled by the user admin.
